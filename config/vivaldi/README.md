first thing first:


1. clone my repository in your git bash or terminal:

```
cd ~ && git clone https://github.com/msalmanrafadhlih/bloodrage-theme.git ~/bloodrage-theme
cd bloodrage-theme
```

2. then make import.sh executable & run it!

```
source ~/import.sh
```
